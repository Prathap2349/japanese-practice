# 🇯🇵 Japanese Writing Practice

A clean and interactive Japanese writing practice web app built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, just one file.

🔗 **Live Demo:** https://prathap2349.github.io/japanese-practice

---

## ✨ Features

### 📝 Writing Practice
- **Hiragana** (ひらがな) — Seion, Dakuten, Handakuten, Combo
- **Katakana** (カタカナ) — Seion, Dakuten, Handakuten, Combo
- **Japanese Numbers / Kanji** (数字)
- Freehand drawing canvas with guide character overlay
- Grid lines for stroke alignment

### 🎨 Drawing Tools
- Multiple pen colors
- Adjustable brush thickness
- Eraser mode
- Clear canvas button
- Guide character toggle (show/hide)

### 🧠 Learning Aids
- Romaji display for each character
- Example word per character
- Audio pronunciation button (Text-to-Speech)
- Stroke order hints

### 🎮 Gamification
- XP system — earn points for every character practiced
- Level progression
- Badges — unlock achievements as you practice
- Daily goal tracker — set and track your daily character goal
- ⏱ Timer Challenge — write as many characters as you can in 60 seconds
- Shuffle mode — randomize character order

### 👤 User System
- Username login with name locking
- Multi-user support (scores saved per user)
- Logout button
- Progress saved locally via `localStorage`

### 🌙 UI & Accessibility
- Dark mode toggle
- Fully responsive — works on mobile, tablet, and desktop
- PWA installable — add to home screen on iOS/Android
- Smooth animations and transitions
- Paper texture aesthetic

## 🚀 Getting Started

No installation needed. Just open `index.html` in any browser.

```bash
git clone https://github.com/Prathap2349/japanese-practice.git
cd japanese-practice
open index.html
```

Or visit the live demo directly: https://prathap2349.github.io/japanese-practice

## 🛠 Tech Stack

| Tech | Usage |
|------|-------|
| HTML5 | Structure & Canvas API |
| CSS3 | Styling, dark mode, animations |
| JavaScript (Vanilla) | All logic, no frameworks |
| Google Fonts | Noto Sans JP, DM Mono, Playfair Display |
| Web Speech API | Audio pronunciation |
| localStorage | User progress & settings |
| PWA Manifest | Installable on mobile |
